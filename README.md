# Prepaid Water Credit System

## Introduction
The **HAVEN PREPAID WATER SYSTEM** is a console-based C application designed to manage and automate utility consumption. It allows administrators to register consumers, monitor real-time water usage, handle prepaid credit tokens, and issue automatic safety alerts.

---

## System Key Features
* **Dual Tariff System:** Automatically separates billing rates for **Domestic** and **Commercial** users.
* **Administrative Security:** Secures core database operations behind a master login PIN (**4545**).
* **Automated Safety Monitoring:** * Triggers a warning if a household balance drops below **5.0 units**.
  * Raises a critical alert if daily usage spikes past **50.0 units** (identifying potential pipe leakages).
* **Maintenance Fee Deduction:** Automatically deducts a fixed **500 UGX** service fee from transactions to fund ongoing network sustainability.
